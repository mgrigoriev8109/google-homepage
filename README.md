Title: Google Homepage Project

Created: 12/1/2020 - 12/9/2020

Motivation: This project was created to gain familiarity working with HtML, CSS, and CSS Flexbox through the creation of a page that looks similarly to Google's Homepage.

Code: The layout of this webpage is done through Flexbox. There are 5 boxes, making up the Header, Google Image, Text Input, Buttons, and Footer, titled box1, box2, ... box5. Box 2, 3, and 4 are located in one main wrapper, which is grown to fill up the entire page between the header and footer. 

Credits: Thank you to the Odin Project and it's community have helped me create this first project.
